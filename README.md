# 🦇 Cave Maze — Projeto de CG

Bem-vindo ao **Cave Maze**, um projeto desenvolvido para a disciplina de **Computação Gráfica**.

Este é um simulador simples de uma exploração dentro de uma caverna labirintica, construído sobre um motor gráfico 3D web.

## 🛠️ Tecnologias

- **Linguagem:** Javascript
- **Bibliotecas e Engine:** GLSL, Vite e WebGLEngine

## 📦 Como Rodar

### Localmente

Ter Node.js instalado. Requerimentos adicionais são nativos web instaláveis via comandos de terminal:

```bash
npm install
npm run dev
```

## 🎮 Controles

- **WASD:** Movimentação.
- **Mouse:** Visão e direção.
- **Esc:** Pausar.
